01 · PROJECT OVERVIEW

项目概览

聚焦智慧养老院连续照护场景，从真实服务痛点出发构建端边云协同的多机器人网络化闭环系统。

从养老服务痛点到系统化解决方案

本作品针对夜间巡护压力大、异常事件预警不足、多任务响应协同困难，以及复杂环境下机器人安全执行不稳定等问题，构建“端—边—云”协同的多机器人网络化闭环系统。

系统由巡检机器人、看护／呼叫机器人、云端协同调度平台、网页监控中心和护工移动端组成，实现异常发现、风险识别、任务编排、现场处置与结果反馈的全过程联动。

异常发现→风险识别→任务编排→现场处置→结果反馈
端边云协同多机器人协作异常行为识别动态任务调度安全自主执行

02 · SYSTEM ARCHITECTURE

系统架构

数据流、任务流与告警反馈流共同形成“感知—汇聚—决策—执行—反馈”的闭环协同链路。

ARCHITECTURE LOGIC

端、边、云协同闭环

端侧现场感知、任务接收、安全执行与结果回传
边侧融合定位、异常识别、风险预测与局部决策
云侧全局状态汇聚、任务编排、动态调度与告警反馈

护理人员终端接收告警、确认处置并参与接管，机器人与人工服务节点在云端统一协调下持续协作。

03 · CORE TECHNOLOGIES

核心技术

围绕可靠感知、异常识别、动态协同与安全执行，形成相互衔接的关键技术体系。

01

异常行为检测

识别当前异常行为，并对短时跌倒风险进行预测。

02

融合定位与可靠识别

结合UWB与多源传感信息，提升复杂环境下定位稳定性。

03

多机器人动态调度

面向动态任务插入与并发事件进行任务排序和能力匹配。

04

安全自主执行

结合全局路径、局部避障与恢复机制实现稳定执行。
